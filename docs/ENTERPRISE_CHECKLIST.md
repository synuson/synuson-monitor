## 1. 사전 준비 단계

### 1.1 요구사항 분석

- [ ] 알림 수신자 및 에스컬레이션 정책 정의
- [ ] SLA 요구사항 정의 (가용성, 응답시간 등)
- [ ] 데이터 보존 기간 결정
- [ ] 사용자 역할 및 권한 정의

### 1.2 인프라 준비
- [ ] 서버 사양 결정 (아래 표 참조)
- [ ] 네트워크 구성 계획
- [ ] 방화벽 정책 수립
- [ ] 백업 정책 수립
- [ ] SSL 인증서 준비

#### 서버 사양 가이드

| 모니터링 규모 | Zabbix Server | Database |
|--------------|---------------|----------|
| ~100 호스트 | 2코어, 4GB RAM | 2코어, 4GB RAM, 50GB SSD |
| ~500 호스트 | 4코어, 8GB RAM | 4코어, 8GB RAM, 200GB SSD |
| ~1000 호스트 | 8코어, 16GB RAM | 8코어, 16GB RAM, 500GB SSD |
| 1000+ 호스트 | 16코어, 32GB RAM | 분산 DB 구성 권장 |
